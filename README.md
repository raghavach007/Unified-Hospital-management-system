This projest made using Spring-boot java, MySQL, HTML, CSS and js
feel free to use this project and modify it according to your wishes
before running this project, in the application.properties file change your database password and username
the pre requirements for this project are java 17 or higher, Node.js and live-server extenstion
if you satisfy the above requirments then use the following command to run the application
command : mvn clean spring-boot:run //(for running the program for the 1st time)
command : mvn spring-boot:run //(from the 2nd time you can use this command)
then open the login.html file and run the live server
now the application is live.
